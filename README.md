1. What is the difference between var, let, and const?
Ans : var is function-scoped and can be redeclared and reassigned.
let and const are block-scoped; let can be reassigned but const cannot be reassigned.

2. What is the spread operator (...)?
Ans : The spread operator (...) is used to expand elements of an array or object into individual values.

3. What is the difference between map(), filter(), and forEach()?
Ans :map() creates a new array by transforming each element, while filter() creates a new array with elements that match a condition.
forEach() only loops through the array and performs an action but does not return a new array.

4. What is an arrow function?
Ans : An arrow function is a shorter way to write functions in JavaScript using the => syntax.

5. What are template literals?
Ans : When using template literals, variables or expressions can be placed inside strings using ${}.
